# Plan: Stack Manager Skill/Agent/Command

**Date:** 2025-12-13
**Status:** COMPLETED
**Objective:** Create stack-manager skill, agent, and command for recommending and bootstrapping app stacks

---

## Architecture

```
/stack-manager (command)
    → stack-manager (agent)
        → stack-manager (skill)
            → ./stacks/*.md (6 stack definitions)
```

---

## Selection Methods (User Chooses)

| Method | Description |
|--------|-------------|
| Interactive wizard | Ask use case, language, complexity - filter stacks |
| Tag-based search | User provides tags, skill finds matches |
| Show all + recommend | List all stacks, highlight best match |
| AI analysis | Analyze project context, suggest stack |

---

## Bootstrap Methods (User Chooses)

| Method | Description |
|--------|-------------|
| Generate files directly | Create pyproject.toml, Dockerfile, etc. in target dir |
| Cookiecutter templates | Use cookiecutter for templated scaffolding |
| Step-by-step guided | Create each file with user confirmation |
| Dry-run first | Show what will be created, then execute |
